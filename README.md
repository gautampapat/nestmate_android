# NestMate 🏠

**NestMate** is a comprehensive, all-in-one Android application designed to simplify shared living and campus life for students and young professionals. It acts as a central hub for finding roommates, managing shared expenses, discovering housing, and fostering a safe, connected community.

## 🌟 Features

* **🤝 Roommate Finder:** Find your perfect living match using our advanced compatibility scorer and dedicated roommate browsing and group features.
* **💸 Shared Finances:** Easily split bills, track shared expenses, set budgets, and view spending insights with your roommates.
* **🏘️ Housing & Listings:** Browse, list, and discover student housing, apartments, and reliable accommodations nearby.
* **🛍️ Peer-to-Peer Marketplace:** Buy, sell, or trade items with others in your community. Create bundles, wanted posts, and chat securely with sellers.
* **🚗 Ride Sharing:** Match with shared rides, schedule trips, and calculate fair splits for your daily commute.
* **🛡️ Health & Safety Hub:** Quick access to emergency contacts, safety features, and a dedicated health services hub.
* **🍽️ Mess & Restaurants:** Browse local mess menus, read restaurant reviews, and discover great places to eat around campus.

## 🛠️ Tech Stack

* **Language:** Kotlin
* **UI Framework:** Jetpack Compose (Modern Declarative UI)
* **Architecture:** MVVM (Model-View-ViewModel)
* **Backend / DB:** Firebase (Authentication, Firestore, Cloud Storage)
* **Image Processing:** Cloudinary API

## 🚀 Getting Started

To run this project locally, you will need to connect it to your own Firebase project:

1. Clone the repository.
2. Create a new project in the [Firebase Console](https://console.firebase.google.com/).
3. Register an Android App and download the `google-services.json` file.
4. Place the `google-services.json` file inside the `app/` directory.
5. Setup Firebase Authentication and Firestore rules.
6. Build and run using Android Studio!

*(Note: Sensitive keys, environment variables, and `google-services.json` are intentionally excluded from this repository for security purposes.)*
